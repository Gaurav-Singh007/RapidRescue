## 📖 Overview

RapidRescue is an emergency response application that enables users to send an SOS request, share their live location, and quickly connect with nearby ambulance services during medical emergencies.

The application is designed to reduce ambulance response time through GPS tracking, AI assistance, and an easy-to-use emergency interface.

---

## ✨ Features

- 🚨 One-Tap Emergency SOS
- 📍 Live GPS Location Sharing
- 🚑 Nearby Ambulance Notification
- 🤖 AI Emergency Assistant (24×7)
- 🏥 Nearby Hospital Information
- 📱 Android & iOS Compatible
- 🔐 Secure User Login
- ⚡ Modern Responsive UI
